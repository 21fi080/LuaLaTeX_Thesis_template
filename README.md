# 日中混植対応 卒論梗概テンプレート (TDU　FI生用)

日本語（MS明朝/ゴシック）と簡体字（中国語フォント）を、同一文書内で混植できるLuaLaTeX用の卒論梗概テンプレートです。

## 概要
従来のpLaTeX環境では難易度の高かった「日本語と簡体字の混在」を、LuaLaTeXと`luatexja-fontspec`を用いることでシンプルに解決しています。
MSフォント（日本語）とFandolフォント（簡体字）を適切に使い分ける設定済みです。

## 動作環境 / 前提条件
本テンプレートを使用するには、以下の環境をご自身で構築している必要があります。

* **TeX環境**: TeX Live (2023以降推奨)
    * コンパイラは **LuaLaTeX** を使用します。
* **エディタ**: VS Code (Visual Studio Code)
    * 拡張機能「LaTeX Workshop」の利用を推奨します。

※ インストール手順については本READMEでは扱いません。ご自身のOSに合わせて環境構築を行ってください。

## 特徴
* **日中混植対応**: `\chn{...}` コマンドで囲むだけで、簡体字フォント（Fandol）に切り替わります。
* **フォント設定済み**:
    * 日本語: MS明朝 (Main), MSゴシック (Bold/Sans)
    * 簡体字: FandolSong (Main), FandolHei (Bold)
    * 欧文: Times New Roman
* **フォーマット準拠**: 指定された余白、フォントサイズ、2段組設定を反映済みです。

## 使い方
1.  `template.tex` を VS Code で開きます。
2.  **LuaLaTeX** でビルド（コンパイル）します。
    * ソースコード冒頭にマジックコメント `% !TEX program = lualatex` を記述済みですので、LaTeX Workshop環境であれば通常通り保存・ビルドするだけで自動判別されます。
